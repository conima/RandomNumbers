==========================================================
Random Numbers Generator
==========================================================

----------------------------------------------------------
Overview
----------------------------------------------------------
Program generates a list of 10,000 numbers in random order each time it is run. Each number in the list must be unique and be between 1 and 10,000 (inclusive).

Added option to set custom input.

----------------------------------------------------------
Setup
----------------------------------------------------------

ASP.NET web application

Language: C#
.NET Framework version: 4.5.2
